# 🚚 Porter Delivery Data Analysis 

## 📊 Project Overview

This project analyzes delivery data to understand factors affecting delivery time and operational efficiency. The goal is to optimize delivery performance and improve customer satisfaction.

---

## 🧹 Data Cleaning

* Handled missing values
* Converted date-time columns
* Created delivery time in minutes
* Removed invalid and negative values

---

## ⚙️ Feature Engineering

* Created `order_hour` and `day_of_week`
* Calculated `delivery_time_minutes`
* Created `within_30_min` flag
* Calculated `partner_load`

---

## 📊 Analysis Performed

* Orders by Hour (Peak Demand)
* Orders by Day
* Category vs Delivery Time
* Market vs Orders
* Market vs Delivery Time
* Partner Load vs Delivery Time
* Items vs Delivery Time

---

## 📈 Dashboard

Two dashboards were created:

1. Demand & Time Analysis
2. Delivery Performance Analysis

---

## 🔍 Key Insights

* Peak hours significantly impact delivery performance
* High partner load leads to delivery delays
* Certain categories take longer to deliver
* Large orders increase delivery time

---

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Data Cleaning Techniques

---

## 📂 Files Included

* Excel file (analysis + dashboard)
* Dashboard screenshots

---

## 🚀 Conclusion

The analysis helps identify operational bottlenecks and improve delivery efficiency through better resource planning.
